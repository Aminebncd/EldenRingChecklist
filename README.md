# Elden Ring — Mega Checklist (MERN)

## quickstart

```bash
cp .env.example .env
pnpm i
pnpm seed
pnpm dev
```

- api: http://localhost:4000
- web: http://localhost:5173
- test user: `test@local` / `test1234`

## scripts
- `pnpm dev` → lance server + web
- `pnpm build` → build des deux paquets
- `pnpm lint` → lint
- `pnpm seed` → insère 3 items + 1 user
